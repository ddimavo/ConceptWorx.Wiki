Эндер-лук (Ender Bow) Выстреливает жемчуг Эндера вместо стрел Зачарования, изменяющие снаряд; Нестрельные зачарования I
Взрывные стрелы (Explosive Arrows) Стрелы взрываются при попадании Нестрельные зачарования III

  • Зачарования, изменяющие снаряд (Bomber, Ender Bow, Ghost) несовместимы друг с другом
  • Нестрельные зачарования - зачарования, не предназначенные для лука/стрел
  • Зачарования из группы "изменяющие снаряд" можно наносить только по одному на лук
  • "Снайпер" совместим со всеми типами зачарований
  • Максимальный уровень влияет на силу/длительность эффекта

# ⛏️ Зачарования для инструментов

Название (Оригинал) Эффект Несовместимо с Макс. уровень Основной предмет Дополнительный предмет
Взрывная добыча (Blast Mining) Добывает блоки взрывами Туннель (Tunnel), Добыча жилы (Veinminer) V
Стеклолом (Glass Breaker) Мгновенно разбивает стекло I
Ускорение (Haste) Даёт эффект Ускорения при добыче III
Удачливый шахтёр (Lucky Miner) Увеличивает опыт с добытых блоков III
Пересадчик (Replanter) Автоматически пересаживает урожай I
Шёлковый сундук (Silk Chest) Сундуки выпадают с содержимым I
Шёлковый спаунер (Silk Spawner) Шанс получить спаунер мобов Плавильщик (Smelter) I
Плавильщик (Smelter) Переплавляет добытые ресурсы Шёлковое касание (Silk Touch), Шёлковый спаунер (Silk Spawner) V
Телекинез (Telekinesis) Лут попадает прямо в инвентарь I
Вальщик леса (Treefeller) Вырубает целое дерево I
Туннель (Tunnel) Добывает блоки в области 1×2/2×2/3×3 Добыча жилы (Veinminer), Взрывная добыча (Blast Mining) III
Добыча жилы (Veinminer) Добывает всю рудную жилу сразу Туннель (Tunnel), Взрывная добыча (Blast Mining) III

Стратегии использования

- Пересадчик + Телекинез = Автоферма
- Ускорение + Удачливый шахтёр = Быстрая прокачка
- Шёлковый сундук + Телекинез = Безопасный перенос лут-сундуков
 
- Взрывная добыча уничтожает лут (не сочетать с Телекинезом)
- Плавильщик несовместим с Шёлковым касанием
- Туннель и Добыча жилы - взаимоисключающие
 
- Кирочка: Добыча жилы + Удачливый шахтёр
- Топор: Вальщик леса + Телекинез
- Мотыга: Пересадчик + Ускорение
- Ножницы: Шёлковый спаунер + Стеклолом

Совет: Создайте отдельные инструменты для разных задач - это повысит эффективность добычи в 3 раза!

# 🎣 Зачарования для рыбалки

Название (Оригинал) Эффект Макс. уровень Основной предмет
Автоподсечка (Auto Reel) Автоматически подсекает рыбу при поклёвке I
Проклятие утопленника (Curse of Drowned) Шанс выловить утопленника вместо рыбы III
Двойной улов (Double Catch) Удваивает количество пойманных предметов III
Речной мастер (River Master) Увеличивает дальность заброса V
Опытный рыбак (Seasoned Angler) Увеличивает количество получаемого опыта III
Выживальщик (Survivalist) Автоматически готовит сырую рыбу I

Стратегии использования

  • Автоподсечка + Двойной улов = Максимальный фарм
  • Опытный рыбак + Речной мастер = Быстрая прокачка
  • Выживальщик + Двойной улов = Запас готовой еды
  • Проклятие утопленника даёт редкие дропы (медные слитки, трезубец)
  • Речной мастер увеличивает шанс поймать глубоководные трофеи
  • Выживальщик экономит время на приготовлении пищи
  • Для фарма ресурсов: Двойной улов + Опытный рыбакДля прокачки навыка: Автоподсечка + Речной мастерДля выживания: Выживальщик + Двойной улов

Cовет: Используйте зачарованную удочку в сочетании с навыком Рыбалка McMMO для максимального бонуса к редким трофеям!

# 🌐 Универсальные зачарования

Название (Оригинал) Эффект Несовместимо с Макс. уровень Основной предмет Дополнительный предмет
Проклятие Износа (Curse of Breaking) Увеличивает износ предмета при использовании Неразрушимость (Unbreaking) III
Проклятие Хрупкости (Curse of Fragility) Блокирует ремонт на точиле или наковальне I
Проклятие Посредственности (Curse of Mediocrity) Ухудшает качество дропа с блоков и мобов III
Проклятие Неудачи (Curse of Misfortune) Обнуляет дроп с блоков и мобов Удача (Fortune), Добыча (Looting) III
Восстановление (Restore) Сохраняет предмет от разрушения (1 раз) III
Привязка к душе (Soulbound) Предмет не выпадает при смерти Проклятие Исчезновения (Curse of Vanishing) I

Особенности проклятий:

  • Проклятие Износа - для баланса мощных предметов
  • Проклятие Хрупкости - защита от кражи ценных вещей
  • Проклятие Неудачи - вызов для хардкорных игроков
  • Восстановление - "последний шанс" для легендарных предметов
  • Привязка к душе - обязательна для ключевых артефактов
  • Проклятия нельзя снять с помощью точила
  • Эффекты проклятий работают даже после ремонта
  • Несовместимые зачарования нельзя нанести вместе

💡 Совет: Комбинируйте Привязку к душе с Восстановлением для полной защиты ваших легендарных предметов!
